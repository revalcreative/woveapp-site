# Wove Site — Image Guide

Drop your image files into the `/images/` folder with these exact filenames.
The site will pick them up automatically.

## Required Images

| Filename | Used On | Size Suggestion | Description |
|---|---|---|---|
| `logo.png` | All pages (nav + footer) | ~200x70px | Your Wove script logo |
| `app-screenshot.png` | Homepage hero | ~600x1200px | Screenshot of the app (like your home screen) |
| `app-store-badge.svg` | Homepage + CTA sections | ~180x60px | Apple "Download on the App Store" badge |

## Notes
- SVG or PNG both work. SVG preferred for icons (crisp at any size).
- Logo should have a transparent background.
- App screenshot should be a clean phone screenshot (no device frame needed — the site doesn't add one).
- If you don't have an icon yet, the dashed placeholder will show instead.
